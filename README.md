# Jukecuke

Run your cukes multi-threaded in JRuby.  Inspired by cukeforker (https://github.com/jarib/cukeforker)

## Installation

Add this line to your application's Gemfile:

    gem 'jukecuke'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install jukecuke
